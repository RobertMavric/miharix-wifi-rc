1. Go to Setup and configure the module
You can access setup any time by opening 192.168.4.1/setup
(or conf,config,admin,administrator,root,nastavi,uredi)

Setup parameters:
  • Pin - Where to connect the servo
  • Trim - Sub Trim - fix center point in Â°
  • Max - maximum trevel in ms
  • Min - minimum travel in ms
  • Fail - failsave position in ms
  • Expo - Expo Rate - reduce center sensitivity
  • Rate - Speed of the servo
  • Reverse - reverse servo movement
  • read EEPROM - reset to last saved state
  • Write RAM - use those till power off
  • Write EEPROM - use those also after power offf
  • Fail. time - Time in s till failsave trigers
  • Mode - Controling interface style
  • RC Name - Name of the RC modell
  • SSID - WiFi name
  • PASS - Password for the WiFi
  • AP - Standalone AccesPoint(AP) or connect to existing WiFi
  • reboot - reboot only
  • reset all - reset module to factory settings
  • set wifi - save WiFi settings
2. Go directly to Control using default settings.
  • For manual factory reset connect GPIO 0 to GND till blinking stops.

  • !! Do not connect any GPIO to 5V(middle pin), it will destroy the chip !!

  • In case of saving wrong credentials for existing WiFi wait 5min to automatic factory reset.
